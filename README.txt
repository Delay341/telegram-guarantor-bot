Телеграм бот-гарант (v3)

Изменения относительно первой версии:
- Кнопка «Мои сделки» показывает список последних сделок (как /mydeals).
- Отправка товара:
  • можно нажать кнопку «Товар отправлен»;
  • либо продавец отправляет сообщение с товаром в чат бота — оно пересылается покупателю, и запускается этап подтверждения.
- Исправлена ошибка Telegram «Can't parse entities: unsupported start tag 'id'» (убраны угловые скобки вокруг id в тексте).
